# Sentences for Audio Generation

This document contains all 240 spelling words with 3 example sentences each.
**Total audio files needed: 720 files (240 words × 3 sentences)**

## Audio File Naming Convention

For each word, create 3 audio files:
- `{word}_sentence1.wav`
- `{word}_sentence2.wav`
- `{word}_sentence3.wav`

Place in: `Resources/Audio/Lisa/sentences/difficulty_{X}/`

## Audio Generation Instructions

### Recommended Tools:
1. **ElevenLabs** - High quality AI voice generation
2. **Google Cloud Text-to-Speech** - Scalable API
3. **Azure Speech Services** - Microsoft's TTS
4. **Amazon Polly** - AWS text-to-speech

### Process:
1. Use Lisa's voice (or closest AI voice match)
2. Generate 720 audio files (240 words × 3 sentences)
3. Export as WAV format (16-bit, 44.1kHz recommended)
4. Name files: `{word}_sentence{1|2|3}.wav` (lowercase word)
5. Place in: `Resources/Audio/Lisa/sentences/difficulty_{X}/`

### Batch Generation Script Example:
```python
# sentences_generator.py
import json
from elevenlabs import generate, set_api_key, save

set_api_key("your-api-key")

# Load sentences from JSON
with open('sentences.json') as f:
    sentences = json.load(f)

for difficulty, words in sentences.items():
    for word_data in words:
        word = word_data['word']
        for i, sentence in enumerate(word_data['sentences'], 1):
            audio = generate(
                text=sentence,
                voice="Lisa",  # or your chosen voice
                model="eleven_monolingual_v1"
            )
            filename = f"Resources/Audio/Lisa/sentences/difficulty_{difficulty}/{word.lower()}_sentence{i}.wav"
            save(audio, filename)
            print(f"Generated: {filename}")
```

### Quality Checklist:
- [ ] Clear pronunciation of the target word
- [ ] Natural speaking pace (not too fast)
- [ ] Appropriate volume levels
- [ ] Kid-friendly tone
- [ ] Correct word emphasis
